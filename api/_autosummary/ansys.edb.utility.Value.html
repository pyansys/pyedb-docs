ansys.edb.utility.Value#

class ansys.edb.utility.Value(val, _owner=None)#

Represents a number or an expression.

Notes

Values can be either constant (e.g. 1, 2.35, 7+0.3i, 23mm) or parametric (e.g. w1 + w2)

if the Value is parametric, it needs _owner set to the object that hosts the variables used. If the owner is Cell or Layout, the expression can reference both Database variables and Cell variables. A better way to create parametric values is to call obj_with_variables.create_value(str) which will automatically set the _owner to the correct object.

Values can be used in expressions with the following operators:
