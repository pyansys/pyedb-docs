insert_zone#

LayerCollection.insert_zone(copy_zone=- 1)#

Insert a new zone.

Parameters
copy_zoneint, optional

If valid the new zone is inserted as a copy of the specified zone, otherwise the new zone is empty

Returns
int

If successful, the id of the newly added zone is returned
