Base class representing a layer.

Get the product property of the layer associated with the given product and attribute ids.

tuple[int, int, int]: Color of the layer.

int: The unique ID of an EDB object.

bool: Determine whether this object exists in EDB.

bool: Flag indicating if the layer is a StackupLayer.

Layer.is_via_layer

bool: Flag indicating if the layer is a ViaLayer.

Layer.layer_id

int: Layer id of the layer.

Layer.layout_obj_type

Layer.locked

bool: Flag indicating if the layer is locked.

Layer.msg

str: Name of the layer.

Layer.top_bottom_association

TopBottomAssociation: Top-bottom association of the layer.

Layer.transparency

int: Transparency value of the layer.

Layer.type

LayerType: Layer type of the layer.

Layer.visibility_mask

int: Visibility mask of the layer.

Layer.zone

int: Zone index associated with owning layer collection.

Layer.zones

list[int]: Zone ids of all zones containing the layer.
